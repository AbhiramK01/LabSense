# Project Abstract

LabSense is an AI-powered online coding lab examination system addressing critical challenges in Computer Science education where students resort to memorizing or copying code since traditional evaluation systems focus solely on final output, failing to assess logic, effort, and error tolerance. The system overcomes limitations of existing methods (MOSS, JPlag, Deckard, PDG, ML-based approaches) that are easily fooled by code variations, miss semantic similarity, or require extensive training. LabSense employs Large Language Models (LLMs) to evaluate code logic and semantics, implementing a multi-component scoring system that assesses code quality (20%), logic similarity using semantic analysis (40%), and test case performance (40%), awarding marks fairly even with different coding styles while providing comprehensive AI-generated feedback. The platform features intelligent question allocation with visualized lab layout mapping assigning unique questions to neighboring systems, real-time tab monitoring, fullscreen enforcement, and comprehensive anti-cheat mechanisms, supporting multiple programming languages (Python, JavaScript, Java, C, C++, Go) through Judge0 Cloud API integration. Built with React/TypeScript frontend and FastAPI/Python backend, LabSense integrates multiple LLM providers (OpenAI, Anthropic, Ollama), implements multi-tenant architecture for scalable institutional deployment, and offers flexible deployment from free Cloudflare Tunnel to production VM setup, providing a comprehensive solution that promotes genuine learning and logical understanding in coding lab examinations.

**Keywords:** AI, Evaluation, Education, Assessment, Programming, Automation, Feedback, Security
